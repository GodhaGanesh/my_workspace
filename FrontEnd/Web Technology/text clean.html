Underline:-

✔we can underline a text by using U tag
Example:- <u>Text</u>
Result :- Text

✔sementic tag of U tag is ins tag
Example:- <ins>Text</ins>
Result :- Text

Bold:-

✔we can make Bold a text by using B tag
Example:- <b>Text</b>
Result :- Text

✔sementic tag of B tag is strong tag
Example:- <strong>Text</strong>
Result :- Text

Italic:-

✔we can make Text Slant by using i tag
Example:- <i>Text</i>
Result :- Text

✔sementic tag of i tag is em tag
Example:- <em>Text</em>
Result :- Text

Strike:-

✔we can Strike a text by using two types of tags they are metioned Below:-

1.s tag :

Example:- <s>Text</s>
Result :- Text

2.strike tag :

Example:- <strike>Text</strike>
Result :- Text

✔sementic tag of s tag is del tag
Example:- <del>Text</del>
Result :- Text

subscript tag :-

✔we can write a text below the base line by using sub tag
Example:- <sub>Text</sub>
Result :- Text

superscript tag :-

✔we can write a text above the base line by using sup tag
Example:- <sup>Text</sup>
Result :- Text

Mark:-

✔we can Highlight a text by using mark tag
Example:- <Mark>Text</Mark>
Result :- Text

small:-

✔we can make a text small by using small tag
Example:- <small>Text</small>
Result :- Text
